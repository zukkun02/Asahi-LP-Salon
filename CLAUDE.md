# Asahi LP Salon — プロジェクトコンテキスト

## 概要
Asahiオンラインサロン（心理学×メンタルウェルネス）のランディングページ。
- デプロイ先: `asahi-lp-1.vercel.app`
- 構成: `index.html` + `style.css`（静的HTML、フレームワークなし）
- リポジトリ: `https://github.com/zukkun02/Asahi-LP-Salon`

## デザインシステム（ui-ux-pro-max スキル適用済み）

| 項目 | 値 |
|------|-----|
| スタイル | Soft UI Evolution（ウェルネス向け、WCAG AA+） |
| カラーパレット | Human Warmth — セージグリーン/ナチュラル系 |
| Primary | `#3D8B5E` (sage-green) |
| Background | `#F5F9F6` |
| Text | `#1A2E22` |
| フォント見出し | Noto Serif JP（Japanese Elegant） |
| フォント本文 | Noto Sans JP |
| シャドウ | 多層シャドウ（ボーダーなし） |
| カード | 上部グラデーションストライプ + 4層シャドウ |

## LP セクション構成（現在）
1. **Hero** — タグライン + 見出し + CTA「サロン参加する」
2. **共感** — 生きづらさチェックリスト（単一カード）
3. **10の特典** — 2列×5行グリッド（番号バッジ付き）
4. **一生物の資産** — 3カード（自己コントロール力 / 幸せ脳 / 人間関係）
5. **料金プラン** — スタンダード ¥5,000 / VIP プレミアム ¥10,000（審査制）+ CTA
6. **Asahiって何者？** — プロフィール + メディア掲載リンク + 実績（大数字） + お客様の声6件
7. **よくある質問** — トグル（`<details>`）アコーディオン、1問目のみ展開
8. **最終CTA** — バイオレットグラデーション背景 + 白CTAボタン
