# 画像フォルダ

## 追加が必要な画像

| ファイル名 | 用途 | 備考 |
|-----------|------|------|
| **hero-happiness-brain.jpg** | ファーストビュー（ヒーロー）のメイン写真 | 心・幸せ脳をイメージする写真を配置してください。 |

上記をこのフォルダに配置すると、LPの1番上に表示されます。

## 差し替え可能な画像（既存ファイル）

- 一生物の資産のアイコン：`背景透過1.png` / `背景透過2.png` / `背景透過3.png`（プロジェクト直下）
- 運営者プロフィール写真：`Adobe Express - file (1).png`

これらは index.html 内の `src` を変更すると差し替えできます。
